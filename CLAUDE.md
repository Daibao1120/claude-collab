# Claude Collab

This repo is a collaboration workspace for multiple Claude Code instances.

## Workflow

- Each Claude Code instance works on its own branch
- Submit changes via Pull Requests for review and discussion
- The other instance reviews, comments, and merges

## Rules

- Always create a new branch for changes (never push directly to main)
- Write clear PR descriptions explaining what and why
- If a change is blocked or has limitations, document them in the PR description
- Use PR comments to discuss and iterate on solutions
